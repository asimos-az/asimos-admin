# Asimos Admin Panel

React (Vite) admin UI for the Asimos mobile app.

## Config

Create `.env` from `.env.example`:

```
VITE_API_BASE_URL=https://asimos-backend.onrender.com
```

## Run

```bash
npm install
npm run dev
```

## Login

Default static super admin:
- Email: `admin@asimos.local`
- Password: `admin1234`

> Change these by setting backend env vars: `ADMIN_EMAIL`, `ADMIN_PASSWORD`, `ADMIN_JWT_SECRET`.
